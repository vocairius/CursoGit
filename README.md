# 🚀 Mi Primer Repo con Git

¡Hola! Bienvenido a mi repositorio de aprendizaje. 👋

Este espacio es mi "laboratorio" personal 🧪 donde estoy poniendo en práctica todo lo aprendido en mi curso de **Git y GitHub**.

---

## 🎯 ¿Qué hay aquí?

Este repositorio documenta mis primeros pasos en el control de versiones. Aquí encontrarás:

* 📜 **Archivos de prueba:** Experimentos con texto y código.
* 🌿 **Manejo de Ramas:** Prácticas de creación, fusión y eliminación de branches.
* ⚔️ **Conflictos:** Simulaciones de conflictos de fusión (Merge Conflicts) y sus soluciones.
* 📝 **Apuntes:** Notas sobre comandos clave.

> [!NOTE]
> El cine es la forma más pura de arte, combina los demás artes dentro de uno solo, por eso es considerado el septimo arte.

## 🛠️ Tecnologías

* **Git:** Control de versiones.
* **GitHub:** Alojamiento remoto.
* **Terminal/Zsh:** Línea de comandos.
* **VS Code:** Editor de código.
